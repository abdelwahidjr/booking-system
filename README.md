# Booking System

This is an Room Booking System using Laravel VueJS.

## Prerequisites

- Git.
- PHP.
- Composer.
- Laravel CLI.
- MySQL.
- A web server like Nginx or Apache.
- NPM Node Package Manager.

## Setup

```
$ git clone 

$ cd booking-system
```

### Install
```

$ composer install 
$ npm install 
$ php artisan project:install 
$ npm run dev 

$ php artisan serve
```
